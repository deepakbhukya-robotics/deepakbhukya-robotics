<h2 align="center">About Me</h2>

I am an **Engineering Management graduate and Mechatronics Engineer** working at the intersection of **robotics, embedded systems, automation, product development, project coordination, data analytics, and engineering design**.

At the University of Maryland Baltimore County, I supported an autonomous hydrogen fuel-cell rover program involving **290+ students and 70+ multidisciplinary teams**. My work included embedded programming, circuit integration, sensor validation, motor control, wireless data transmission, design reviews, technical documentation, troubleshooting, competition planning, and prototype-readiness evaluation.

My engineering experience includes:

- Autonomous robotics and embedded control systems
- Arduino programming and sensor integration
- Electrical and electronics system design
- IoT monitoring and web-based engineering dashboards
- Machine learning and data analytics
- CAD, simulation, and electromagnetic modeling
- Product discovery, QFD, CTQs, market sizing, and risk analysis
- Project planning, WBS, CPM, Agile, Scrum, and technical documentation

I approach engineering with a product-focused mindset: understand the problem, identify user requirements, define measurable performance targets, evaluate technical risks, build reliable systems, validate results, and document decisions clearly.
